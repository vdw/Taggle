Usage

Include jQuery at your document's footer

Include Taggle

Create the following html structure 


  





  tag 1
  tag 2
  tag 3
  tag 4

Initialize Taggle using the following two ways 

// Default way
$(document).ready(function() {
  $('.my-article').taggle();
});
